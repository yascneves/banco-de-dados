Banco de dados - Aula 2 - Prof. Sávio

Dado: é qualquer elemento em sua forma bruta
Informação: é um agrupamento de dados que fazem sentido
Fato: é um conjunto de dados relacionados
Metadados:

Coletando os DADOS, podemos processá-los e dessa forma é gerada uma INFORMAÇÃO. Analisando as informações, é possível definir o que é FATO.

- O que é Banco de Dados (BD)?
Representação digital de dados interrelacionados representando informações sobre um domínio específico.

- Minimundo:
O "mundo" das informações que serão processadas (ex.: todos os produtos de um mercado)

- 4 operações de um BD:
Inserção;
Remoção;
Atualização;
Consulta.

- Abstração de dados:
omissão de informações do usuário para uma melhor usabilidade
